5. Examples

In the following simple usage examples of Joystick Gremlin will be shown.These should give you an idea of what is possible and illustrate the use of the UI.

Remapping an axis

  1. Select the tab of the device for which to remap the axis
  2. Select the desired axis by either clicking on it or moving the physical axis itself
  3. Click the Add as the Remap action is always selected by default
  4. Select the vJoy device and axis to remap to from the drop downs

Remapping a button

  1. Select the tab of the device for which to remap the button
  2. Select the desired button by either clicking on it or pressing the physical button itself
  3. Click the Add as the Remap action is always selected by default
  4. Select the vJoy device and button to remap to from the drop downs

Remapping a hat

  1. Select the tab of the device for which to remap the hat
  2. Select the desired button by either clicking on it or moving the physical hat itself
  3. Click the Add as the Remap action is always selected by default
  4. Select the vJoy device and hat to remap to from the drop downs

Custom axis response curve

Remapping an axis and assigning it a custom response curve is rather straight forward and involves the following steps:
  1. Select the tab of the device you wish to crete an axis remapping for
  2. Select the desired axis by either clicking on it or moving the physical axis itself
  3. Open the action drop down menu and select the Remap action
  4. Click the Add button to add the Remap action
  5. Next select the Response Curve action from the drop down menu and add it
  6. Now you can modify the response curve to your liking, detailed usage information is given in Section 2.3 but the most important functions are:
    • Double left click adds a new control point
    • Single left click selects a control point
    • To move a control point simply drag it around
    • To delete a control point select it and then hit the Del key
