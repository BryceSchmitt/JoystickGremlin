Overview

The main goal of Joystick Gremlin is to allow you to perform various actions in response to pressing keys on the keyboard or using a joystick.

There are four main concepts that Joystick Gremlin uses to organize and describe configurations:

  1. Input, keyboard keys, joystick axis, buttons and hats all are inputs
  2. Action, a single action executed when an input is used
  3. Mode, a collection of actions can be grouped together into a mode
  4. Profile, the file containing all modes and their associated actions

Creating a New Profile

As such creating a new profile involves the following steps:

  1. Create a new empty profile
  2. Create at least one mode
  3. Add actions to different inputs
  4. Save the profile
  5. Run the profile

1. Create a new empty profile

To create a new profile click on File > New Profile.

2. Create a new mode

Before you can start adding actions to various inputs you need to create a new mode. This is achieved via Tools > Manage Modes. This opens a new window, clicking on the Add Mode button allows you to add a new mode.

3. Add actions to different inputs

Once you have added a new mode you can assign actions to the various inputs by clicking on the appropriate button in the UI or pressing the desired button, moving the desired axis on the joystick. Once the input for which you want to add an action is selected you can select the type of action to add from the dropdown list and then press the Add button. Depending on what type of action was added different settings can be changed.

4. Save the profile

To save your new profile click on File > Save Profile which will prompt you for the location and name of the file in which to save your new profile.

5. Run the profile

Now you can run your profile by pressing the gamepad icon in the menu bar, which turns green when Joystick Gremlin is running.
